Linux distribucije

Linux distribucija je operativni sustav napravljen iz kolekcije software-a, koji su bazirani na Linux kernelu. Korisnici Linuxa uobičajeno dobivaju svoj operativni sustav tako što preuzimaju jednu od Linux distribucija koje su dostupne za razne sustave računala. Tipična Linux distribucija kompromizira Linux kernel, GNU alate, libary-e i dodatni software kako bi sposobnost odgovarati zahtjevima mnogih korisnika.

Primjeri:

Ubuntu je open source operativni sustav za računala, to je Linux distribucija bazirana na Debian arhitekturi. Najčešće je u upotrebi na osobnim računalima, ali je isto tako popularan u izvedbi kao mrežni server. Ubuntu također postoji za tablete i smartphone uređaje uz „Ubuntu Touch“ edition Izdaje ga Canonical Software koji mu isto tako daje i komercijalnu podršku.

Debian je operativni sustav sastavljen samo od besplatnog software-a, koji je većinom pod GNU General Public licencom i „zapakiran“ od strane pojedinaca koji sudjeluju u „Debian projectu“. Debian Project je prvi put najavljen 1993 od strane Ian Murdock-a, Debian 0.01 je u javnost pušten 15.9.1993., a prva stabilna verzija Debiana izašla je tek 1996. Kao jedan od najranijih operativnih sustava baziran na Linux kenelu, odlučeno je kako će se Debian nastaviti razvijati otvoreno i besplatno (open source) u „čast“ GNU projektu. Debian ima pristup online repozitoriju koji sadrži više od 50000 software paketa što ga ujedno čini operativnim sustavom sa najviše software paketa.
